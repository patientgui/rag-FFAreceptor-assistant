# RAG-FFA4

Assistant RAG pour interroger des documents scientifiques sur les récepteurs FFA/GPCR.

## Fonctionnalités
- chargement de PDF et DOCX
- embeddings locaux
- recherche sémantique avec FAISS
- réponse locale ou via LLM local

## Lancer le projet
```bash
pip install -r requirements.txt
python RAG.py
